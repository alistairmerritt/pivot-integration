# Security Policy

## Scope

The Pivot integration runs entirely within your local Home Assistant instance. It does not introduce any cloud services, telemetry, or external network connections.

## Reporting a vulnerability

If you discover a security vulnerability in the Pivot integration, please **do not open a public issue**. Instead, report it privately by emailing the maintainer (contact via GitHub profile) or using [GitHub's private vulnerability reporting](https://github.com/alistairmerritt/pivot-integration/security/advisories/new).

Include:
- A description of the vulnerability
- Steps to reproduce or proof of concept
- Any relevant integration version or Home Assistant version details

You will receive a response as soon as reasonably possible. Confirmed vulnerabilities will be addressed in a patch release, and the reporter will be credited (unless anonymity is requested).

## Supported versions

Only the latest release of the Pivot integration is actively maintained. If you are running an older version, update via HACS before reporting.
